#Información programa
```
Opciones:
      --help            Muestra ayuda                                 [booleano]
      --version         Muestra número de versión                     [booleano]
  -i, --identificacion  Indetificación de persona
                                              [cadena de caracteres] [requerido]
  -m, --mostrar         Mostrar información almacenada
                                         [booleano] [requerido] [defecto: false]
```
